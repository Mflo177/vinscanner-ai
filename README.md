# VIN Scanner AI - Mobile Computer Vision Application

A real-time optical character recognition (OCR) mobile app using Google ML Kit for text detection, the device's camera, and data management features.

## ‚ú® Features:

### Core Functionality
- **Real-time OCR**: Camera-based text detection using ML Kit
- **Pattern Validation**: Alphanumeric code verification (17-character format)
- **Barcode Support**: Detects both plain text and barcoded codes
- **Data Management**: CRUD operations on captured entries
- **Metadata Support**: Custom notes, location tags, status tracking
- **CSV Export**: Generate spreadsheet files with one tap
- **Email Integration**: Share exports via native email client
- **Offline-First**: No internet required for core functionality
